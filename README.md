# 📈 Options Learning Platform

> A free paper options trading simulator built for classrooms.

![License](https://img.shields.io/badge/license-MIT-green)
![Cost](https://img.shields.io/badge/hosting%20cost-%240%2Fmonth-267C2C)
![Devices](https://img.shields.io/badge/works%20on-Chromebook%20%7C%20iPad%20%7C%20Desktop-0a6fba)

Students practice real options strategies with a paper account, live P&L tracking, and an assignment risk simulator — all in a browser, no install, no broker account required. Teachers get class codes, a student leaderboard, and one-click account resets.

---

## ✨ Features

### For Students
- **21 options strategies** — from covered calls and cash-secured puts to iron condors, straddles, jade lizards, and more
- **Real P&L math** — seller and buyer formulas calculated correctly, with a progress bar showing % of max profit reached
- **Assignment Risk Simulator** — inline panel showing both scenarios (expires worthless vs. gets assigned) with break-even analysis
- **Dividend tracking** — log ex-date, pay date, and dividends collected per position
- **Stock holdings tab** — track the shares behind your covered calls and CSPs
- **Summary & Analytics** — win rate, P&L by strategy, realized vs. unrealized breakdown
- **$25,000 paper account** — starting balance set by teacher, editable per student

### For Teachers
- **Class codes** — generate a 6-character code (e.g. `FIN3A7`), students join instantly on signup
- **Live leaderboard** — see all students ranked by account value with trade counts and win rates
- **Read-only student view** — click any student to see their full dashboard without editing anything
- **Account reset** — reset one student or the entire class with a single click
- **Multiple classes** — manage different periods or sections independently
